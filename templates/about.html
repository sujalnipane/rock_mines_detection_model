About This Project

Sonar-based Rock vs Mine Classification System

🎯 Overview

Ye project sonar signal data analyze karke predict karta hai ki underwater object ek rock hai ya mine. UCI Sonar dataset use kiya gaya hai, jisme 60 frequency-based features hote hain har sample ke liye.

🧠 Model Training

Model ko Google Colab par train kiya gaya hai using scikit-learn. Data preprocessing, feature scaling, aur model evaluation Colab notebook mein perform kiya gaya, fir trained model ko pickle file (.sav) ke form mein save kiya gaya.

⚙️ Tech Stack

  • Model Training: Google Colab, scikit-learn, pandas, numpy
  • Backend: Flask (Python)
  • Frontend: HTML, CSS, JavaScript
  • Deployment: Render

🚀 Workflow

Dataset load → preprocessing → model training & evaluation (Colab) → model export as pickle file → Flask app banaya jo model ko load karke real-time predictions deta hai → frontend se user input leta hai → Render par deploy kiya gaya final web app ke roop mein.
